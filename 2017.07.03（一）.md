    b.currentLanguage

  。「未登入」狀態  
    Case 1：User自己切換語言 －＞ 「a」為最高權重 (update「b」)  
    Case 2：進入系統，系統背後 run 「default 語言」 －＞ 以「b」為最高權重，最後是「c」(此case不會有「a」)  
    　　User今天第一次開啟網頁 or Page reload  


  。「登入ing」  
    Case 1：登入時會去抓取到新的「c」，這時候應該要已「c」為最高權重，「b」應該被update。　(此case不會有「a」)
